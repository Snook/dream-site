Setting up a custom HTML signature in Outlook Express

1) First create or locate an HTML file that contains your desired signature. Make sure that you now the location where you saved the file.


2) Now we are ready to set it up as a signature in Outlook Express.
Select Tools|Options.




3) Select the Signatures Tab, then New. Click on the name that comes up (in this case Signature #1) and then on Rename and name it something that makes sense to you.

NOTE: If you want the Signature on all messages, check the box under Signature settings.




4) Now to the bottom of that same screen, click in the button by File, then on the Browse button. Go to where you saved the html file you made earlier and click on it to open it. In "Files of Type" select HTML as shown in the example. Select your file and click Open. Then click OK at the bottom.




5) Alright, now you have it set up as a Signature.
